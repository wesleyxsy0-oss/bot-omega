# Bot ÔMEGA – Análise de Prescrição Fiscal

Web app para análise automática de CDAs e processos de execução fiscal.

## Como usar
1. Faça upload de um CSV com as colunas exigidas.
2. Clique em "Analisar".
3. Baixe o resultado com semáforo e recomendação.

## Colunas necessárias
- numero_cda
- data_fato_gerador (AAAA-MM-DD)
- data_inscricao (AAAA-MM-DD)
- data_citacao (AAAA-MM-DD)
- ultima_movimentacao (AAAA-MM-DD)
- valor (opcional)
